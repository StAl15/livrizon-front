## Стек
<p align="left">
Frontend - ReactJS (Typescript + TailwindCSS)
  <br/>
Backend - Java Spring Boot
   <br/>
CRM - Sanity
  
## Запуск проекта
  
frontend
```
npm install 
npm start
```
  
sanity
```
sanity start
sanity manage //for detail info
```
